System Control Center

ERP စနစ်၏ နောက်ကွယ်ဒေတာများကို အရန်သင့်သိမ်းဆည်းခြင်း၊ လစာစာရွက်များကို အီးမေးလ် ပေးပို့ခြင်းနှင့် ဘဏ္ဍာရေးနှစ်သစ်စာရင်းဖွင့်ခြင်းများကို ဤနေရာတွင် ဆောင်ရွက်နိုင်ပါသည်။

Spreadsheet Excel Backup System

ကျောင်း၏ ERP စနစ် တစ်ခုလုံးရှိ အချက်အလက်များ လုံးဝ ပျက်စီးပျောက်ဆုံးမှု မရှိစေရန်အတွက် goldeneduprivateschool@gmail.com သို့ Excel (.xlsx) Backup ဖိုင်အဖြစ် ချက်ချင်း ပေးပို့ရန် ခလုတ်ကို နှိပ်နိုင်ပါသည်ရှင်။

Monthly Staff Payslip Dispatcher (Resend API)

Full Time နှင့် Part Time ဝန်ထမ်းများထံသို့ ယခုလအတွက် လစာဖြတ်ပိုင်းများကို အီးမေးလ်ဖြင့် ပေးပို့ပါမည်။ Resend.com API ကို အသုံးပြု၍ တစ်လလျှင် တစ်ကြိမ်သာ အဆင်ပြေစွာ ပို့ဆောင်ပေးမည် ဖြစ်ပါသည်။

Run EOY Fiscal Year Reset (ဘဏ္ဍာရေးနှစ်သစ် စာရင်းဖွင့်ခြင်း - Admin သီးသန့်)

မတ်လအကုန် ဧပြီလအစ (March to April) ဘဏ္ဍာရေးနှစ်ကူးပြောင်းမှုအတွက် သမိုင်းကြောင်း စာရင်းဟောင်းများကို စနစ်တကျ ပိတ်သိမ်းပါမည်။ လုပ်ဆောင်ချက်စတင်သည်နှင့် စာရင်းဟောင်းဒေတာများကို Google Drive ထဲသို့ **အလိုအလျောက် Backup ပွားယူ သိမ်းဆည်းမည်** ဖြစ်ပြီး၊ ၎င်းနောက် စာအုပ်များမှ ရရန်ကျန်/ပေးရန်ကျန် ချေးငွေများနှင့် ရန်ပုံငွေများကို dynamic တွက်ချက်ကာ **Opening လက်ကျန်အသစ်များအဖြစ် စာရင်းသစ်ပြန်လည် တည်ဆောက်ပေးမည် ဖြစ်ပါတယ်။**

⚠️ သတိပေးချက်- ဤလုပ်ငန်းစဉ်သည် ယခင်နှစ်ဒေတာဟောင်းများကို ဖျက်ဆီးပြီး စာရင်းသစ်ဖွင့်လှစ်မည်ဖြစ်သဖြင့် လှည့်လည်ပြင်ဆင်ခွင့် ရှိမည်မဟုတ်ပါ။ Admin သာ လုပ်ဆောင်ခွင့် ရှိပါသည်။
